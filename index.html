🎓 Optimizing Recruitment: Machine Learning in Hiring

This research, approved for presentation at IEEE AIRC 2025, explores predictive hiring models to classify candidates and enhance recruitment efficiency, emphasizing fairness and reducing biases.

🔍 Approach:

  • ✅ Developed predictive hiring models using algorithms like CatBoost and Random Forest.
  • ✅ Conducted feature engineering to identify key hiring factors such as education level and personality scores.
  • ✅ Implemented bias mitigation techniques to ensure fairness in hiring decisions.

✨ Results:

  • 🚀 Achieved 95% accuracy with CatBoost, identifying the most suitable candidates.
  • 🚀 Reduced biases in recruitment by implementing fairness-focused adjustments.
  • 🚀 Streamlined recruitment processes, improving efficiency and decision-making.

Why it matters: This project demonstrates how machine learning can revolutionize recruitment, ensuring equitable and data-driven hiring decisions.

Selected for presentation at IBAC 2025, this project focuses on leveraging machine learning models to optimize hiring processes, reduce biases, and enhance recruitment efficiency.

🔍 Research Approach:

  • ✅ Built predictive hiring models using CatBoost, Random Forest, and SVM.

✨ Results:

  • 🚀 Enhanced decision-making with transparent and interpretable models.

📦 Enhancing Operational Efficiency: Inventory Management

Leveraging advanced machine learning and time-series forecasting methods, this project optimized inventory management processes, leading to cost reductions and improved customer satisfaction.

🔍 Approach:

  • ✅ Applied SARIMA, Gradient Boosting, and XGBoost for accurate demand forecasting.
  • ✅ Performed extensive feature engineering and hyperparameter tuning to optimize models.
  • ✅ Integrated predictive models with real-time inventory systems for seamless operations.

✨ Results:

  • 🚀 Identified SARIMA as the most accurate model for demand prediction.
  • 🚀 Reduced stockouts and overstocking, leading to a 15–20% inventory cost reduction.
  • 🚀 Implemented Economic Order Quantity (EOQ) and Just-In-Time (JIT) strategies for operational efficiency.

Why it matters: This project showcases how machine learning can revolutionize inventory management, ensuring scalability and efficiency across supply chains.
